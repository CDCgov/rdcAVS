# Create Drive folders into another folder in parallel

Create Drive folders into another folder in parallel

## Usage

``` r
drive_mkdir_parallel(names, target_folder)
```

## Arguments

- names:

  `list` A list of folder names.

- target_folder:

  `dribble` Dribble of the target folder.

## Value

`dribble` A dribble of the recently created folders.

# Copy template folder to the ZS folder

Copy template folder to the ZS folder

## Usage

``` r
drive_cp_zs_template_parallel(template_dribble, zs_drive_folders)
```

## Arguments

- template_dribble:

  `dribble` A dribble object for the template sheet.

- zs_drive_folders:

  `dribble` A dribble of zones de sante folders.

## Value

`dribble` A dribble containing the template files of a campaign.
